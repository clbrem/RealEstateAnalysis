Installation
------------
(using pip):
From the root folder:
1) Install virtualenv:
   $ pip install virtualenv
2) make a virtual environment directory 
   $ virtualenv ./.venv 
3) Activate the virtual environment:
   $ source ./.venv/bin/activate
3) run 
   $ pip install -r requirements.txt
